# Local Loop Bengaluru

Local Loop Bengaluru is a GitHub Pages-ready web platform prototype for discovering cafés, restaurants, bakeries, street food, and neighborhood institutions around Bengaluru.

## Live site

The site is published with GitHub Pages from the `main` branch.

## What is included

- Static map experience centered on Bengaluru
- Leaflet map with OpenStreetMap tiles
- Filter chips for local gems, institutions, budget eats, cafés, bakeries, late night, vegetarian, and street food
- Curated sample places with coordinates and neighborhood context
- Responsive layout for desktop and mobile

## Deployment

This repository is configured for GitHub Pages. The app is intentionally static so it can be served directly from the repository root without secrets or a server.
